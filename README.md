# Проект SRE

Простое веб-приложение на Go с CI/CD-пайплайном.

## Возможности
- Веб-интерфейс для управления задачами.
- REST API для управления задачами.
- Контейнеризация с помощью Docker.
- CI/CD-пайплайн с использованием GitHub Actions.
